# Vanguarda_2003

Réplica inicial da página da TV Vanguarda a partir do snapshot da Wayback Machine.

## Origem do snapshot
- URL arquivada: `http://www.vanguarda.tv/`
- Snapshot Wayback: `https://web.archive.org/web/20030920043455/http://www.vanguarda.tv/`
- Data/hora do arquivamento (UTC): `2003-09-20 04:34:55`

## Estrutura inicial
- `index.html`
- `branca.htm`
- `indice.htm`
- `top.htm`
- `conteudo.htm`
- `css/tv.css`
- diretórios de assets: `img/` e `js/`

## Observação
Nesta etapa, foram preservados os HTML/CSS do snapshot e criada a árvore de arquivos com os mesmos nomes referenciados no código original (incluindo `img/`, `apps/`, `jornalismo/`, `comercial/` e `cobertura/`) para manter compatibilidade de paths.
